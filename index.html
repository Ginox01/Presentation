JobFinder

In primis , il progetto è visionabile al seguente : Link.

Questo progetto si avvale della API di The Muse , il progetto inoltre è sviluppato con il metodo della "single page application" grazie all'utilizzo di Angular.
Come potete notare , nella Home è presente un form dove è possibile filtrare le varie ricerce per località , livello e mestiere.
Il form mette a display una serie di risultati utili dove è possibile selezionare direttamente l'offerta di lavoro o la pagina dedicata all'azienda

Per quanto concerne la gestione degli errori , se nel form vengono immessi valori errati ( ad esempio un valore negativo nel campo pagina) , la pagina metterà a display un messaggio di errore che informa l'utente della anomalia.

Project UMS Commerce

Il progetto è scaricabile al seguente Link : UMS_Commerce

Questo progetto è un piccolo sistema CRUD , si avvale di Angular , Json-server e Bootstrap , quindi , per avviare il progetto bisogna installare prima i pacchetti e poi visionare il server con il commando dal prompt : json-server --watch db.json

Nella pagina dedicata al database , nella colonna Azioni , possiamo trovare tre bottoni :

  • Bottone rosso : elimina Utente

  • Bottone verde : aggiorna Utente

  • Bottone Blu : aggiungi Utente nel carrello

È possibile anche creare un nuovo utente dal Form

I campi del form necessitano di particolari condizioni , come ad esempio la lunghezza minima del nome o la presenza della @ nel campo mail

Note App

Il progetto è visionabile al seguente link : App Note

Questo progetto è stato realizzato esclusivamente con HTML , CSS e Javascript

Classica applicazione di sticky notes , con le seguenti funzionalità :

  • Eliminazione note : è possibile eliminare le note semplicemente trascinandole verso l'icona del cestino

  • Creazione nota : Cliccando il bottone posto in basso a destra della pagina è possibile accedere all'editor di testo per creare una nuova nota

  • Organizzazione note : È possibile organizzare la priorità delle note trascinando la nota che si desidera spostare con quella dove occupa attualmente la posizione desiderata
